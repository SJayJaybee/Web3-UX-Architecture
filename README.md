## 🔍 Case Studies & Live Prototypes

## 🔴 Module 1: Legacy DEX UX Audit (Mobile Bottlenecks)
An objective teardown of traditional DeFi mobile viewports. This audit isolates specific UI vulnerabilities—such as touch-target violations (sub-44px) and viewport data compression—that actively bleed Total Value Locked (TVL) due to user abandonment.

* **Diagnostic Focus:** WCAG Guidelines, Cognitive Overload Reduction, Mobile Viewport Rendering.
* **Strategic Proposal:** Implementation of a "Retail-First Abstraction Layer" to hide advanced routing data behind intent-driven UI cards.

### 🟢 Module 2: Frictionless Web3 Onboarding (Interactive Flow)
A 3-screen interactive prototype demonstrating a fully abstracted Web2-to-Web3 retail checkout. This architecture bypasses traditional wallet generation, allowing users to authenticate via Google and purchase crypto via Apple Pay/Debit in two clicks.

* **Tech Stack Simulated:** Account Abstraction (ERC-4337), Social Login, Fiat On-Ramps.
* **Design Philosophy:** Zero-jargon, institutional-grade dark mode (`#0A192F`), and high-contrast call-to-action pathways designed for maximum conversion.
